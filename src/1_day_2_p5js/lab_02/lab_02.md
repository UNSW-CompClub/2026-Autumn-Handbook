# DIY!

## You can do it!

Over the course of the last two days, you've learnt a whole lot about p5.js. It's time to apply those skills! Using the p5.js reference [found here](https://p5js.org/reference/), make something cool and interesting using one or more functions not discussed in our lectures! The cooler and more unique your project, the more points you'll get. Try and go for a masterwork!

## Project ideas

- Extend MS Paint with more features
- A self portrait
- A fancier game like Snake
- A simulation (e.g. Fire, Boid's algorithm for flocking birds)
